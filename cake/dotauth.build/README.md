# dotauth.build (Cake build helpers)

## Purpose
Contains build helpers and tasks used by the repository's Cake build scripts. This project groups reusable build logic and task implementations used by the `build.sh`/`build.ps1` scripts in the repository root.

## Quickstart
- Location: `cake/dotauth.build/dotauth.build.csproj`
- Build (from repo root): `dotnet build ./cake/dotauth.build/dotauth.build.csproj`

## Notes
This project is intended as part of the repository build tooling and usually isn't run directly. See the repository root `README.md` and the `build.sh` / `build.ps1` scripts for build automation instructions.
