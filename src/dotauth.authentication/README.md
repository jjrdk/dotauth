# dotauth.authentication (Authentication library)

## Purpose
Contains authentication logic and helper services shared across DotAuth components. This is typically a class library used by server projects.

## Quickstart
- Location: `src/dotauth.authentication/dotauth.authentication.csproj`
- Build: `dotnet build ./src/dotauth.authentication/dotauth.authentication.csproj`

## Notes
This is a library; run and test via consuming projects or unit tests in the `tests/` folder.
