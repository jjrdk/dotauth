# dotauth.uma.web (UMA web UI)

## Purpose
Web UI components and endpoints for UMA workflows and user consent management.

## Quickstart
- Location: `src/dotauth.uma.web/dotauth.uma.web.csproj`
- Build: `dotnet build ./src/dotauth.uma.web/dotauth.uma.web.csproj`

## Notes
This project complements `dotauth.uma` and provides web-facing pages for UMA features.
