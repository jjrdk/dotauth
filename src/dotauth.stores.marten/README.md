# dotauth.stores.marten (Marten-backed storage)

## Purpose
Implementation of storage using Marten (PostgreSQL + .NET document DB) for DotAuth's persistence needs.

## Quickstart
- Location: `src/dotauth.stores.marten/dotauth.stores.marten.csproj`
- Build: `dotnet build ./src/dotauth.stores.marten/dotauth.stores.marten.csproj`

## Notes
Requires PostgreSQL. Marten-specific configuration and schema migration steps may be required before running the server.
