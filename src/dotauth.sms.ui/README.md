# dotauth.sms.ui (SMS UI components)

## Purpose
User-interface components and pages related to SMS verification flows. Likely part of the web UI projects that render SMS prompts and inputs.

## Quickstart
- Location: `src/dotauth.sms.ui/dotauth.sms.ui.csproj`
- Build: `dotnet build ./src/dotauth.sms.ui/dotauth.sms.ui.csproj`

## Notes
This project may be consumed by web UI projects or packaged as part of the overall UI.
