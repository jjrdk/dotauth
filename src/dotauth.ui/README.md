# dotauth.ui (Web UI components)

## Purpose
Web UI components and pages for DotAuth that render user-facing authentication flows and pages.

## Quickstart
- Location: `src/dotauth.ui/dotauth.ui.csproj`
- Build: `dotnet build ./src/dotauth.ui/dotauth.ui.csproj`

## Notes
This project contains UI views and assets used by the server. See related `dotauth.sms.ui` and `dotauth.uma.web` projects for additional web interfaces.
