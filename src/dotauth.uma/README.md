dotauth.uma (UMA extensions)

Purpose
-------
Provides User-Managed Access (UMA) extensions and support for DotAuth, implementing UMA-specific flows and endpoints.

Quickstart
----------
- Location: `src/dotauth.uma/dotauth.uma.csproj`
- Build: `dotnet build ./src/dotauth.uma/dotauth.uma.csproj`

Notes
-----
See `src/dotauth.uma.web` for web integration and acceptance tests under `tests/` for behavior verification.
