# dotauth.shared (Shared primitives)

## Purpose
Shared types, DTOs, and utilities used across multiple DotAuth projects. Contains models and helpers to keep interfaces consistent.

## Quickstart
- Location: `src/dotauth.shared/dotauth.shared.csproj`
- Build: `dotnet build ./src/dotauth.shared/dotauth.shared.csproj`

## Notes
This is a class library referenced by server and client projects.
