# dotauth (Core server components)

## Purpose
Core server components and entry points for the DotAuth server. 
This project contains the main server host and shared startup logic used by other server variants.

## Quickstart
- Location: `src/dotauth/dotauth.csproj`
- Build: `dotnet build ./src/dotauth/dotauth.csproj`
- Run (if applicable): `dotnet run --project ./src/dotauth/dotauth.csproj`

## Notes
See repository root `README.md` and the `Dockerfile*` files for common deployment configurations.
