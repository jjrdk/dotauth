# dotauth.tool (Tooling project)

## Purpose
Contains tooling utilities and CLI helpers used by DotAuth developers (e.g., migration scripts, admin tools).

## Quickstart
- Location: `src/dotauth.tool/dotauth.tool.csproj`
- Build: `dotnet build ./src/dotauth.tool/dotauth.tool.csproj`
- Run (if it is a CLI): `dotnet run --project ./src/dotauth.tool/dotauth.tool.csproj`

Check the project source to see available commands and usage examples.
