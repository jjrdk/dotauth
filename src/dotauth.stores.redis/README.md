# dotauth.stores.redis (Redis-backed store/caching)

## Purpose
Provides Redis-backed storage or caching layers for DotAuth, used for session storage, caching, or ephemeral state.

## Quickstart
- Location: `src/dotauth.stores.redis/dotauth.stores.redis.csproj`
- Build: `dotnet build ./src/dotauth.stores.redis/dotauth.stores.redis.csproj`

## Notes
Requires a running Redis instance and configuration via environment variables or appsettings.
