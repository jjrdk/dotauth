# dotauth.sms (SMS provider library)

## Purpose
Provides SMS sending capabilities and abstractions used by the DotAuth project for SMS-based verification and notifications.

## Quickstart
- Location: `src/dotauth.sms/dotauth.sms.csproj`
- Build: `dotnet build ./src/dotauth.sms/dotauth.sms.csproj`

## Notes
This project likely relies on provider-specific configuration (API keys, endpoints). Keep secrets out of source control and provide them via environment variables.
