# dotauth.client.maui (MAUI client sample)

## Purpose
A .NET MAUI client demonstrating integration with DotAuth. Useful as a reference mobile/desktop client implementation.

## Quickstart
- Location: `src/dotauth.client.maui/dotauth.client.maui.csproj`
- Build: `dotnet build ./src/dotauth.client.maui/dotauth.client.maui.csproj`
- Run (platform-specific): requires MAUI workloads and platform SDKs (Android, iOS, Windows/macCatalyst) and an appropriate IDE (Visual Studio).

## Notes
This project requires additional platform tooling not included in the repository. See MAUI documentation for environment setup.
