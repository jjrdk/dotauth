# dotauth.client (Client library)

## Purpose
Client-side helpers and SDKs for applications integrating with DotAuth. Provides convenience methods for token acquisition and interaction with the server.

## Quickstart
- Location: `src/dotauth.client/dotauth.client.csproj`
- Build: `dotnet build ./src/dotauth.client/dotauth.client.csproj`

## Notes
This is a library project intended to be consumed by applications. See `src/dotauth.client.maui` for a MAUI client example.
